# Bookly.html
Bookly — Modern Book Discovery UI

Bookly is a sleek and modern book discovery interface built with clean HTML and CSS.
The project recreates a dark-themed Figma design, featuring curated collections, category chips, featured books, and a responsive layout.

-  Features

   Modern dark UI with gradient backgrounds

   Curated collections and editor’s picks

   Search bar for books, authors, or ISBN

   Category chips (Science, Fantasy, Classics, etc.)

   Featured book cards with ratings

   Popular picks grid

   Fully responsive — adapts to tablet & mobile

   Accessible structure (semantic HTML, aria labels)

   Tech Stack

- HTML5

- CSS3 (modern gradients, flexbox, grid)

Google Fonts — Inter

No JavaScript frameworks (pure frontend)
